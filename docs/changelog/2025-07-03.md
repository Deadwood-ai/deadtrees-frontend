## Technical Details

- **Data Source**: Gap-filled VIIRS phenology data (VNP22Q2v001) stored in `v2_metadata` table as JSONB
- **Data Structure**: 365 daily values (0-255 scale) representing growing season probability 2013-2022
- **Activity Classifications**:
  - Off Season (🟤): 0-85 values - dormant/winter season
  - Transition (🟡): 86-170 values - transitional periods
  - Peak Season (🟢): 171-255 values - peak growing season
- **Resolution**: 10km × 10km pixel resolution
- **Color Gradient**: Linear interpolation from #F5F5F5 (light grey) to #388E3C (dark green)
- **Positioning**: Day-of-year calculations using 366-day timeline for leap year compatibility

## Impact

Users can now immediately assess whether datasets were captured during optimal growing season for dead tree detection, improving data quality evaluation and analysis planning. Enhanced tooltips provide scientific context for better understanding of phenology data.
